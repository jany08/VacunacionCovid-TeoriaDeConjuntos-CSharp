# VacunacionCovid-TeoriaDeConjuntos-CSharp
Sistema desarrollado en C# que utiliza operaciones de conjuntos (unión, intersección y diferencia) para clasificar ciudadanos según su estado de vacunación.
# Campaña de Vacunación COVID-19

## Descripción
Este proyecto fue desarrollado en lenguaje C# utilizando teoría de conjuntos para simular una campaña de vacunación contra el COVID-19.

Se creó un conjunto ficticio de 500 ciudadanos y se generaron subconjuntos de ciudadanos vacunados con Pfizer y AstraZeneca.

## Objetivos del programa
El sistema permite obtener los siguientes listados:

- Ciudadanos no vacunados.
- Ciudadanos que han recibido ambas dosis.
- Ciudadanos que solo han recibido la vacuna Pfizer.
- Ciudadanos que solo han recibido la vacuna AstraZeneca.

## Conceptos aplicados
- HashSet en C#
- Unión de conjuntos
- Intersección de conjuntos
- Diferencia de conjuntos
- Teoría de conjuntos
